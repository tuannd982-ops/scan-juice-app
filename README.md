# Juice Scan App

## Thông tin sinh viên

* Họ và tên: Phạm Minh Tuấn
* Mã sinh viên: 23810310308

## Mô tả

Ứng dụng React Native mô phỏng màn hình Scan sản phẩm nước ép.

Các chức năng:

* Màn hình scan sản phẩm
* Khung scan 4 góc
* Animation scan line
* Card hiển thị thông tin sản phẩm

## Công nghệ sử dụng

* React Native
* Expo
* Expo Router
* Ionicons
